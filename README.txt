4º missão - DevMedia Carreira Programador Front-End

A proposta desse projeto é criar uma página web estilizada e que exibe listas. 
Teremos um arquivo HTML que conterá a estrutura da página da imobiliária e um arquivo CSS 
que conterá os estilos para mudar a aparência da página.

Ao criar esse projeto, será colocado em prática os seguintes conceitos:

- Tags básicas do HTML
- Meta tags HTML
- Listas no HTML
- Seletores CSS

Algumas propriedades de cor foram alteradas se comparado ao projeto sugerido pela DevMedia.
